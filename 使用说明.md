### 🛠️ 如何配置并使用？（面向 VPS 用户）

尽管服务支持国内 IPv6 直连，但官方推荐的使用场景仍是**通过海外 VPS 作为客户端**来享受服务，这能提供更稳定的使用体验。

#### **准备工作**

1.  拥有一台**海外服务器（VPS）**，系统建议：Ubuntu 20.04+、Debian 10+、CentOS 7+ 等主流 Linux 发行版。
2.  服务器拥有公网 **IPv4 地址**。
3.  具备基本的 Linux 命令行操作能力。

#### **第一步：添加服务器到白名单**

访问 [用户仪表盘](https://icmp9.com/user/dashboard) -\> **放行管理** -\> **IPv4 白名单页面**，添加您的 **VPS 的公网 IPv4 地址**。

> **💡 小贴士：** 虽然您无法添加国内 IPv4，但您的海外 VPS IPv4 地址是必须添加的。如果您使用国内 IPv6 直连，请在对应页面添加您的 IPv6 地址。

#### **第二步：在服务器上安装客户端**

通过 SSH 连接到您的海外 VPS，并使用以下一键安装脚本：

```bash
bash <(curl -L -s api.icmp9.com/icmp9.sh)
```

#### **第三步：按照安装向导操作**

脚本将引导您完成配置：

  * 选择节点地区。
  * 输入您的 **API Key**（从用户中心首页复制）。
  * 配置隧道参数（通常保持默认）。
  * 服务将自动启动并设置开机自启。

#### **第四步：验证连接**

检查用户中心仪表盘的节点状态是否显示为「**在线**」，并查看流量统计是否正常更新。
